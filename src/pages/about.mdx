Hi, my name is Andrea Vouk (He/His) and I'm a professional software developer based in Italy.

I'm self-taught through books and the internet, my interests range from SysAdmin/SRE/virtualization stuff to
backend development, IoT and Kubernetes. I like deep dives into low-level stuff and learning new stuff.

Big fan of Apache Kafka, Kubernetes, Proxmox, QEMU and distributed systems in general.

## OSS projects

Some personal projects that might be of interest:

- [andreavouk.com](https://github.com/anvouk/andreavouk.com): this website source code, built with Astro.
- [lua-resty-jwt-verification](https://github.com/anvouk/lua-resty-jwt-verification): a JWT validation library written in
Lua for the OpenResty platform (It's basically nginx on steroids).
- [fakeopoly (WIP)](https://github.com/anvouk/fakeopoly): A FOSS recreation of the notorious, and very much copyrighted, game.
Still very much a work in progress, will make a public website once done. One day...
